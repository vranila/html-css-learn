CSS overflow Property

The overflow property controls how content is handled when it exceeds the boundaries of its container.

Overview

Values of the overflow property:

  • visible – Default. Content spills outside the box.
  • hidden – Extra content is clipped (not visible).
  • scroll – Always adds scrollbars (even if not needed).
  • auto – Adds scrollbars only when necessary.
