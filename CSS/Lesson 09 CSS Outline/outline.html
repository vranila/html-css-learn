CSS Outline

An outline is drawn around an element — outside its border — to make it stand out.

1️⃣ What is an Outline?

A CSS outline is a line drawn outside an element's border. It does not affect the element’s size or box dimensions.

This element has a black border and a green outline of 10px.

div {
  border: 2px solid black;
  outline: 10px solid green;
}

Note: Outlines differ from borders — outlines sit outside the border and can overlap nearby elements.
