CSS float Property

The float property determines how elements are positioned relative to their container — typically used to let text wrap around images or align blocks side-by-side.

Example 1: float: right;

The image on the right uses float: right;. The surrounding text wraps around the image dynamically. Resize the browser to see how the text adjusts to the image placement.
