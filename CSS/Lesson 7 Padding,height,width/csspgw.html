CSS Padding, Height & Width

Understand how padding and dimensions work in the CSS box model.

1️⃣ CSS Padding

The padding property creates space inside an element, between its content and border.

This element has 70px padding on all sides. 
div {
  padding-top: 50px;
  padding-right: 30px;
  padding-bottom: 50px;
  padding-left: 80px;
}

2️⃣ Box-Sizing: Border-Box

Normally, padding adds to an element’s total width. Using box-sizing: border-box; keeps the element size fixed while padding adjusts internally.

Fixed width of 300px even with 25px padding. 
div {
  width: 300px;
  padding: 25px;
  box-sizing: border-box;
}

3️⃣ CSS Height and Width

The height and width properties control an element’s content area. Padding, border, and margin expand the total space taken.

Height: 200px, Width: 50% 
div {
  height: 200px;
  width: 50%;
  background-color: powderblue;
}
