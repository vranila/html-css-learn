CSS display: inline-block - Detailed Example

The display: inline-block property combines the behavior of inline and block-level elements.

Example 1: display: inline vs inline-block vs block

The example below shows how display: inline, display: inline-block, and display: block behave differently:

Inline Inline-Block Block
  • Inline: Elements flow inline and ignore width/height properties.
  • Inline-block: Elements flow inline but respect width/height and vertical spacing.
  • Block: Elements start on a new line and take up the full width.
