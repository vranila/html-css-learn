CSS Box Model

Understand how every HTML element is structured in layers — content, padding, border, and margin.

1️ What is the CSS Box Model?

In web design, every HTML element is considered a rectangular box. The CSS Box Model defines how these boxes are structured and how they interact with each other.

  • Content: The inner area where text, images, or other elements appear.
  • Padding: Clears space around the content, inside the border. Transparent.
  • Border: Wraps around the padding and content. Visible and stylized.
  • Margin: Creates space outside the border, separating the element from others.

This box has all four layers — content, padding, border, and margin.

div {
  width: 300px;
  border: 15px solid green;
  padding: 50px;
  margin: 20px;
}

2️ Box Model Calculation

When you set width and height in CSS, it only affects the content box. To get the actual size of the element, include padding and border.

Box Model Example

Width calculation:

  • Content width = 320px
  • Left + right padding = 10px + 10px = 20px
  • Left + right border = 5px + 5px = 10px
  • Total width = 320 + 20 + 10 = 350px

Height calculation:

  • Content height = 50px
  • Top + bottom padding = 10px + 10px = 20px
  • Top + bottom border = 5px + 5px = 10px
  • Total height = 50 + 20 + 10 = 80px
div {
  width: 320px;
  height: 50px;
  padding: 10px;
  border: 5px solid gray;
  margin: 0;
}

Note: Margin adds space around the element but is not included in the element’s total size.
