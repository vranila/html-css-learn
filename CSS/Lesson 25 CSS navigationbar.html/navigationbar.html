CSS Navigation Bars

Having an easy-to-use navigation is essential for any website. Navigation bars help users quickly move between sections of a site.

Vertical Navigation Bar

Example of a simple vertical navigation bar:

  • Home
  • News
  • Contact
  • About

Horizontal Navigation Bar

Example of a horizontal navigation bar:

  • Home
  • News
  • Contact
  • About

Code Explanation

A navigation bar is built using an unordered list (<ul>) with list items (<li>) containing links (<a>).

<ul>
  <li><a href="default.asp">Home</a></li>
  <li><a href="news.asp">News</a></li>
  <li><a href="contact.asp">Contact</a></li>
  <li><a href="about.asp">About</a></li>
</ul>

Then, you remove default list styling and paddings to clean up the layout:

ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
}
  • list-style-type: none; – Removes bullet points
  • margin: 0; – Removes default browser margins
  • padding: 0; – Removes default browser paddings

Note: This base code is used for both vertical and horizontal navigation bars, depending on how you structure and style the list.
