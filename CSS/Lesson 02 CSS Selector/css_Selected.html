CSS Selectors

CSS Selectors define which HTML elements the styles should apply to. They are the foundation of every CSS rule.

1. Element Selector

The element selector targets all HTML elements of a specific type.

p {
  color: blue;
}

This paragraph is styled using the element selector (p).

2. ID Selector

The ID selector targets a single element with a unique id attribute, using the # prefix.

#unique-heading {
  color: green;
  text-decoration: underline;
}

This heading is styled using an ID selector.

3. Class Selector

The class selector targets one or more elements with a specific class attribute, using the . prefix.

.highlight {
  background-color: yellow;
  font-weight: bold;
}

This is a normal paragraph.

This paragraph uses the .highlight class.

4. Universal Selector

The universal selector (*) applies styles to all elements on the page.

* {
  box-sizing: border-box;
}

Used for global reset or applying a universal property.

5. Grouping Selector

The grouping selector lets you apply the same styles to multiple elements by separating them with commas.

h2, h3 {
  color: #0066cc;
}

This is a level 2 heading

This is a level 3 heading

6. Descendant Selector

The descendant selector targets elements nested within another element.

div em {
  color: red;
}

This emphasized text inside a div will be red.

Summary of Selectors

  • Element Selector: Targets all of a specific HTML tag (e.g., p).
  • ID Selector: Targets an element with a specific id (e.g., #header).
  • Class Selector: Targets elements sharing a class (e.g., .button).
  • Universal Selector: Targets all elements (e.g., *).
  • Grouping Selector: Combines multiple selectors (e.g., h1, h2, p).
  • Descendant Selector: Targets elements nested within another (e.g., div p).
