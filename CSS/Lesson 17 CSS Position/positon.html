CSS position Property

The position property defines how an element is positioned on the web page.

1️⃣ position: static

This is the default positioning. Elements appear in normal document flow.

This <div> element has position: static;

2️⃣ position: relative

Relative elements can be moved from their normal position using top, left, right, or bottom.

This <div> has position: relative; (moved 30px right)

3️⃣ position: absolute

Absolute elements are positioned relative to the nearest ancestor with positioning.

This <div> has position: relative;
This inner <div> has position: absolute;

4️⃣ position: fixed

A fixed element stays in place relative to the viewport, even when scrolling.

Fixed Box (bottom-right corner)

5️⃣ position: sticky

Sticky elements act like relative until you scroll past them — then they “stick”.

This box sticks to the top when you scroll!

Scroll down to see the sticky effect in action.

6️⃣ Positioning Text on an Image

You can use position: absolute; to overlay text on top of an image.
