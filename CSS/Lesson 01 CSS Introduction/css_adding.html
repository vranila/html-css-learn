How to Add CSS

When a browser reads a stylesheet, it formats the HTML document according to the rules defined in the CSS. There are three ways to add CSS to a web page:

  1. External CSS
  2. Internal CSS
  3. Inline CSS

1. External CSS

With an external stylesheet, you can define styles in a separate .css file and link it using the <link> tag inside the <head> section.

/* style.css file */
body {
  background-color: lightblue;
}

h1 {
  color: navy;
  text-align: center;
}
    
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

Explanation: The browser loads the CSS file (style.css) and applies the styles globally to the HTML page.

2. Internal CSS

With internal CSS, the style rules are defined inside the <style> tag within the <head> section of the HTML document.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: linen;
    }
    h1 {
      color: green;
      text-align: center;
    }
  </style>
</head>
<body>

  <h1>This text is styled using internal CSS.</h1>


</body>
</html>

Example Output:

This text is styled using internal CSS.

3. Inline CSS

With inline CSS, you apply styles directly to an individual HTML element using the style attribute.

<h1 style="color:blue; text-align:center;">
  This is a blue heading
</h1>

<p style="color:red;">
  This is a red paragraph.
</p>

Example Output:

This is a blue heading

This is a red paragraph.

Summary

  • External CSS – ideal for applying consistent styles across multiple pages.
  • Internal CSS – used for single-page styling or page-specific adjustments.
  • Inline CSS – best for quick fixes or unique element-level styling.
