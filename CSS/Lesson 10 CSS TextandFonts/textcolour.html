CSS Text Color

Mastering color properties for clean, readable design

Introduction

CSS provides several properties for styling and formatting text. Color plays a key role in readability, aesthetics, and user engagement.

The color Property

The color property is used to define the color of text. It can be defined using:

  • Color names – for example: red
  • HEX values – for example: #ff0000
  • RGB values – for example: rgb(255, 0, 0)

Text and Background Color

The background-color property defines the background color of an element. Use it along with color to ensure strong contrast for accessibility and readability.

Light Background

Background: lightgrey | Text: blue

Dark Background

Background: black | Text: white

Accent Background

Background: blue | Text: white

Accessibility Note

High contrast between text and background is crucial for users with visual impairments. Avoid color combinations that reduce legibility, such as red text on green backgrounds.
