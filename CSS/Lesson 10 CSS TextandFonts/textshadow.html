CSS Text Shadow

Enhance typography with depth, contrast, and visual dimension

1. Introduction

The text-shadow property adds shadows to text. It can take up to four values:

  • Horizontal offset — distance of the shadow from the text (X-axis).
  • Vertical offset — distance of the shadow from the text (Y-axis).
  • Blur radius (optional) — the softness of the shadow.
  • Color (optional) — color of the shadow.

Syntax: text-shadow: horizontal vertical blur color;

2. Basic Shadows

Simple Shadow Effect

Example: text-shadow: 2px 2px;

White Text with Dark Shadow
