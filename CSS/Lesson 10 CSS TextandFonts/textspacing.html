CSS Text Spacing

Master text spacing properties for better readability and design precision

1. Introduction

CSS provides multiple properties to control spacing within and between text elements. Proper spacing enhances readability, creates hierarchy, and gives your content a polished, professional appearance.

  • text-indent — controls the indentation of the first line.
  • letter-spacing — adjusts spacing between characters.
  • line-height — defines spacing between lines of text.
  • word-spacing — adjusts spacing between words.
  • white-space — defines how white spaces are handled.

2. Text Indentation

The text-indent property controls the indentation of the first line in a text block. You can use positive or negative values.
