CSS Text Transformation

Control text capitalization using the text-transform property

1. What is Text Transformation?

The text-transform property is used to change the case of text without altering the original HTML content. It’s useful for enforcing consistent capitalization in headings, labels, or UI text.

2. Available Values

  • none: No transformation — text appears as written in the HTML.
  • capitalize: Transforms the first character of each word to uppercase.
  • uppercase: Transforms all characters to uppercase.
  • lowercase: Transforms all characters to lowercase.
