Mastering text alignment, direction, and vertical positioning in CSS

Text Alignment

The text-align property is used to control the horizontal alignment of text. It determines whether text aligns to the left, right, center, or stretches evenly across the container (justify).

Text Align Last

The text-align-last property controls how the last line of a text block is aligned.

This paragraph has its last line aligned to the right. Text-align-last: right;

This paragraph has its last line aligned to the center. Text-align-last: center;

This paragraph has its last line justified, just like the rest. Text-align-last: justify;

Text Direction

The direction property defines the writing direction of text — left-to-right (LTR) or right-to-left (RTL). The unicode-bidi property ensures the text direction is correctly applied, especially in multilingual layouts.
