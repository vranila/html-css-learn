CSS z-index Property

The z-index property controls which elements appear on top when elements overlap.

Example 1: Image Behind Text

This is a heading

Because the image has a z-index of -1, it appears behind the text.

Example 2: With z-index

A positioned element with a higher z-index always appears above one with a lower value.

Black box (z-index: 1)
Gray box (z-index: 3)
Green box (z-index: 2)

Example 3: Without z-index

If no z-index is specified, the stacking order follows the HTML source order.

Black box (no z-index)
Gray box (no z-index)
Green box (no z-index)
