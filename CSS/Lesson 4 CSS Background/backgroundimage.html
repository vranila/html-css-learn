CSS Background Image Demonstration

Default Background Repeat

The background image repeats both vertically and horizontally by default.

Repeat Horizontally

The image repeats only across the x-axis using background-repeat: repeat-x;.

No Repeat

Only one instance of the image is displayed with background-repeat: no-repeat;.

Positioned Background

The background is aligned to the top-right corner using background-position: right top;.

Fixed Background Attachment

The image stays fixed while the rest of the content scrolls using background-attachment: fixed;.

Shorthand Property Example

This background uses the shorthand property:
background: #fff url('tree.jpg') no-repeat right top;
